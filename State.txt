The application should store its state in files in the application data folder.

APP_NAME: sharedHome

Application data folder: <first choice> if not possible: <second choice>
Linux, Darwin: $XDG_DATA_HOME/APP_NAME OR $HOME/.local/share/APP_NAME
Windows: %APPDATA%/APP_NAME or FAIL

Application config folder: 
Linux, Darwin: $XDG_CONFIG_HOME/APP_NAME
